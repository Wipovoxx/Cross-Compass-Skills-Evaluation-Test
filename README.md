# Cross Compass Skills Evaluation Test

This repository contains the deliverable for the Cross Compass software engineer skills evaluation. It implements a desktop image processing application that applies HSV colour adjustments and sharpness/blur effects to batches of images.

## Quick start

Requirements: [uv](https://docs.astral.sh/uv/getting-started/installation/)

```
cd image-processor
uv sync
uv run image-processor
```

For full setup details, features, and architecture see the [image-processor README](image-processor/README.md).
